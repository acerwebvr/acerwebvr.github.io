  • Author:Anthony Prum

View VR project

Lazy Sunday Simulator was made for a project in my production class with Karen Vanderborght for Integrated Digital Media. We were to create a portrait with no face in VR with an assigned VR creation tool. This project was a technical exercise to get us acquainted with working in a three dimensional space and how to direct a viewer through it.
